# AnjotoMap

Proyecto con Frontend (Vue 3 + Vite) y Backend (Node.js + Express).

## Cómo usar Backend ⚙️

```sh
cd backend
```
1. instalar dependencias
```sh
npm install
```
2. ejecutar backend
```sh
node index.js
```
## Cómo usar el Frontend 💻 

```sh
cd frontend-antojomap
```
1. instalar dependencias
```sh
npm install
```
2. ejecutar frontend
```sh
npm run dev
```
